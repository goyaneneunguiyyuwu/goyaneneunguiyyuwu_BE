## 1. Installation

```bash
$ npm install
```

## 2. running docker container
```bash
$ npm run compose
```

## 3. Running the app

```bash
# development(watch mode)
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
